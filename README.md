# UNO Game

Number-based UNO card game deployed on Render.

## Features
- Play against 1-3 AI bots
- Number-based cards instead of images
- Special cards: +2, +4, block, reverse, wild
- Sound effects and settings

## Deployment
- **Platform**: Render.com
- **Start Command**: `npm start`
- **Node Version**: >=14.0.0

## How to Play
1. Select number of bots
2. Click cards to play them
3. Match color or number
4. First to empty hand wins!
